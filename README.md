# Real Capita Website Scaffold

A clean, beginner-friendly Next.js + TypeScript project structure for a modern corporate website.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Project idea

This scaffold is organized so you can:
- find each feature quickly
- change content from `src/data`
- reuse layout and UI components
- add new pages without breaking old code
